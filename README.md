## Extend it yourself!

Run the following commands to install, build and start the app:

```sh
$ npm install
$ npm start
```

Run the following command to create an app for macOS:

```sh
$ npm run package
$ mv ./dist/Netflix-darwin-x64/Netflix.app /Applications
```
